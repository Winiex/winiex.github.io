Octotree：在 Chrome 里用树状的方式浏览 Github 的文件

Jul 7, 2016

一直以来，Github 的文件浏览功能都不能够满足我的日常使用需求，直到今天我发现了 Octotree 这款 Chrome 浏览器插件。这款插件能够在 Github 的项目页面的左边提供一个树形菜单，让用户能够更方便地浏览项目内的文件，从而避免每次浏览都需要刷新整个页面。
